VeloTime 🏁

La piattaforma definitiva per trasformare le tue uscite Strava in sfide competitive reali.

React + Vite Supabase Backend Strava API

Panoramica del Progetto

VeloTime permette ai ciclisti di connettere il proprio account Strava, importare le attività recenti e convertirle in "Eventi" o "Sfide". Gli altri utenti possono partecipare a queste sfide e confrontare i propri tempi su segmenti specifici, scalando la Hall of Fame globale.

Architettura del Sistema

            graph TD
                User((Utente)) --> Frontend[React SPA / Vite]
                Frontend --> Auth[Supabase Auth]
                Frontend --> DB[(PostgreSQL / Supabase)]
                Frontend --> Functions{Edge Functions}
                Functions --> Strava[Strava API OAuth2]
                Functions --> DB
                DB --> Realtime[Supabase Realtime Feed]
                Realtime --> User

Flusso di Integrazione Strava (OAuth2)

            sequenceDiagram
                participant U as Utente
                participant F as Frontend
                participant S as Strava API
                participant E as Edge Function (Exchange)
                participant DB as Database

                U->>F: Clicca "Connetti Strava"
                F->>S: Redirect a Strava Auth (scope: read,activity)
                S-->>U: Chiede Autorizzazione
                U->>S: Autorizza
                S-->>F: Redirect a /callback?code=AUTH_CODE
                F->>E: Invia AUTH_CODE + JWT Utente
                E->>S: POST /oauth/token (con Client Secret)
                S-->>E: Restituisce Access/Refresh Tokens
                E->>DB: Upsert Tokens (Tabella strava_tokens)
                E-->>F: Successo!
                F->>U: Mostra Attività Importate

Funzionalità Chiave

🔄 Strava Sync

Sincronizzazione automatica delle attività. Gestione intelligente del Refresh Token ogni volta che l'utente accede.

🏆 Hall of Fame

Classifica globale dinamica basata su un sistema di punteggio: 100pt vittoria, 10pt partecipazione, bonus velocità.

⚡ Social Feed

Feed in tempo reale che mostra le nuove sfide create e i record battuti, usando Supabase Realtime.

📍 Segment Tracking

Estrazione automatica dei segmenti Strava durante la creazione dell'evento per sfide mirate sulle salite.
